# 🎯 AI Resume Analyzer

A Python-based NLP tool that analyzes how well a resume matches a job description.

## 💡 What it does
- Extracts text from your resume (PDF)
- Cleans and processes text using NLP
- Compares resume with job description using TF-IDF and Cosine Similarity
- Shows a match percentage score
- Identifies missing keywords from the job description

## 🛠️ Tech Used
- Python
- PyPDF2
- NLTK
- Scikit-learn
- Streamlit

## 🚀 How to Run
1. Clone this repository
2. Install dependencies
3. Run the app

## 📦 Install Dependencies
pip install -r requirements.txt

## ▶️ Run the App
streamlit run app.py
